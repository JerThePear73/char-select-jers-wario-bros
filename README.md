# [CS] Wario Bros.

What's more enticing than saving royalty? Getting rich baby! 

3 characters with custom movesets that get stronger as you pick up coins! Wario, Waluigi, and Captain Syrup.

![CS Wario Bros.](images/title.png)

# Moves:

Wario:

	(B) = Shoulder Bash
	GP + (B) = GP cancel
	GP + (A) = Corkscrew
	Slide + (Z) = Roll
	Swims slower
	Jumps slightly lower

Waluigi:

	(B) = Shoulder Bash
	(B) in air + neutral stick = Arial Shoulder Bash
	GP + (B) = GP cancel
	GP + (A) = Corkscrew
	Swims faster
	Jumps slightly higher

Syrup:

	(B) = Sword Slash
	(B) in air + neutral stick = Sword Chop
	GP + (A) = Sword Tornado
	Dive + (Z) = Elegant Landing
	Hold (A) in water = Fast Swim
	Special cannon movement
